vim-auto-colorscheme
====================

Automatically change the color scheme in the regular expression.

setup in your vimrc

e.g.)
    let g:auto_colorscheme_default = 'nevfn'
    let g:auto_colorscheme_config = {
    \     '~/\.vim' : 'evening',
    \     '\\\\win-file-server\\' : 'desert',
    \ }
